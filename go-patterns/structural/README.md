# Structural

Patterns for composing types and wrapping behaviour in Go.

| Pattern | Description |
|---------|-------------|
| [Proxy](./proxy) | Interface proxy with caching and access control |
| [Decorator](./decorator) | Wrapping concrete types to add behaviour |
| [Adapter](./adapter) | Bridging incompatible interfaces |
| [Middleware Chain](./middleware-chain) | Composable HTTP/gRPC middleware with panic recovery |
