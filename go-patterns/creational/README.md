# Creational

Patterns for flexible and testable object construction in Go.

| Pattern | Description |
|---------|-------------|
| [Functional Options](./functional-options) | Self-referential function options for configurable constructors |
| [Singleton](./singleton) | Thread-safe single instance with `sync.Once` |
| [Builder](./builder) | Step-by-step construction of complex objects |
| [Factory](./factory) | Interface-based factory for polymorphic creation |
