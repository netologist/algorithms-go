# Behavioural

Patterns for orchestrating behaviour and communication between components.

| Pattern | Description |
|---------|-------------|
| [Fan-Out / Fan-In](./fan-out-fan-in) | Parallel workers processing from a shared channel |
| [Pipeline](./pipeline) | Chained stages connected by channels with cancellation |
| [Observer](./observer) | Publish-subscribe with typed event channels |
| [Iterator](./iterator) | Custom iterators with `for range` support (Go 1.22+) |
| [Strategy](./strategy) | Pluggable algorithm selection via interface |
