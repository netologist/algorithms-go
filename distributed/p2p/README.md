# Peer-to-Peer

Decentralised communication patterns for peer-to-peer networks.

| Pattern | Description |
|---------|-------------|
| [Gossip Protocol](./gossip-protocol) | Epidemic-style information dissemination with fanout |
| [Chord DHT](./chord-dht) | Distributed hash table with O(log N) lookups and finger tables |
| [Epidemic Broadcast](./epidemic-broadcast) | SI model epidemic spread with dead node handling |
