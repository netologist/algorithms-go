# Communication

Patterns for exchanging messages between distributed components.

| Pattern | Description |
|---------|-------------|
| [Scatter-Gather](./scatter-gather) | Fan-out requests, aggregate responses with timeout |
| [Pub-Sub](./pub-sub) | Topic-based publish-subscribe with filtering |
| [Request-Reply](./request-reply) | Synchronous request-response over message channels |
| [gRPC Patterns](./grpc-patterns) | Unary, server-streaming, client-streaming, interceptors |
