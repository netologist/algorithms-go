# Observability

Patterns for monitoring, tracing, and understanding distributed system
behaviour in production.

| Pattern | Description |
|---------|-------------|
| [Metrics](./metrics) | Counters, gauges, histograms with Prometheus-style exposition |
| [Health Check](./health-check) | Liveness and readiness probes with configurable checks |
| [Heartbeat](./heartbeat) | Fault detection via periodic pings and failure thresholds |
| [Distributed Tracing](./distributed-tracing) | Span-based tracing with context propagation |
| [Structured Logging](./structured-logging) | Structured log entries with levels, fields, and correlation IDs |
